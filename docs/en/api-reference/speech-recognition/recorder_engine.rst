Recorder Engine
===============

The Recorder Engine API is a set of functions to facilitate voice recording. The API is integrated with :doc:`esp_vad`, providing options to enable and disable VAD to control the incoming audio stream. The Recorder Engine also includes possibility to encode the audio stream using :doc:`AMR or AMRWB <../codecs/amr_codecs>` formats.
